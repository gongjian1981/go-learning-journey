# Go Learning Journey

This repository documents my 14-day journey of learning Go (Golang) from scratch, focusing on building a solid foundation in backend development.

The goal of this project is not just to learn syntax, but to gain hands-on experience by building real-world features such as RESTful APIs, database integration, and basic authentication.

---

## 🚀 What I'm Learning

- Go fundamentals (syntax, structs, slices, maps)
- Concurrency (goroutines, channels)
- Error handling and interfaces
- Building REST APIs using Gin
- Database integration with MySQL and GORM
- Basic authentication (JWT)

---

## 🛠 Tech Stack

- Go (Golang)
- Gin (web framework)
- GORM (ORM)
- MySQL
